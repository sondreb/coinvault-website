coinvault

Coinvault is a none custodian HD wallet for coins based on the Bitcoin protocol, with support for Stratis (STRAX) and Cirrus (CRC). Coinvault also keeps your identities and keys safe, while allowing you to communicate securely and authenticate with web apps.

A Stratis Decentralized Accelerator proposal

Stratis and Cirrus wallet

We make a bid to the Stratis Decentralized Accelerator for funding to develop a none custodian wallet.

The wallet will allow users to store, send, receive, setup coldstaking, create tokens, swap tokens, vote on proposal and generaly interact with smart contracts on top of the Stratis mainchain and sidechains.

All the features in one place

No need to run a full sidechain node to interact with smart contracats

What we place to deliver:

  • HD wallet, one key all coins
  • Send and receive on Cirrus and Stratis
  • Setup and track coldstaing coins
  • Create tokens on Cirrus
  • Swap token with opdex
  • Master nodes can easy vote on SDA proposals
  • All your NFTs in one place
When new contracts are successfully voted on Cirrus coinvault can add support to interact with such contracts.

Technical proposal

We propose to build a wallet UI for the stratis network, however to interact with the Stratis and Cirrus blockchains there need to be a rich and powerfull blockchain indexers also knowen as blockchain explorers. Our work will involve not just building the UI but also exploreres to allow searching and quiering blockchain data.

Wallet

Use HD technology to generate mnemonic to easy backup and recover your wallet.

Store, send, and receive STRAX and CRC in one place.

Smart contracts

Making smart contract on Cirrus accesible without running a fullnode.
- Issue and store tokens on Cirrus.
- Interact with the Opdex protocol to swap token issued on Cirrus.
- Enable master nodes to vote on SDA proposals.
- Create and store NFTs. (pending NFT contracts are available)
- Coldstaking, stake Strax safely with external staking pools

Indexer

Blockcore already has an indexer for stratis we will improve it and also adapt it to Cirrus and create apis to interact with smart contracts on Cirrus.

Explorer

Blockcore explorer will be extended to Stratis and Cirrus, users will be able to quiery to blockchain for information.
